# Security Policy

## Supported Versions

Currently, only the latest `main` branch and the most recent preview/stable releases are actively supported with security updates.

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

Instead, please report them using **GitHub's Private Vulnerability Reporting** feature:
1. Go to the **Security** tab of this repository.
2. Click **Report a vulnerability** (under Vulnerability reporting).
3. Provide details of the issue and submit.

You should receive a response acknowledging the report quickly, and we will work with you privately to verify and fix the issue.
