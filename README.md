# FlatRedBall2

A modern, lightweight 2D game engine built on MonoGame and KNI.

## Overview
FlatRedBall2 is the next generation of the FlatRedBall game engine, offering two backend targets depending on your platform needs:
- **FlatRedBall2.MonoGame** (.NET 10.0): The desktop target.
- **FlatRedBall2.Kni** (.NET 8.0): The browser target (via Blazor WASM) and future mobile targets.

## Packages
The engine is available via NuGet:
- [`FlatRedBall2.MonoGame`](https://www.nuget.org/packages/FlatRedBall2.MonoGame)
- [`FlatRedBall2.Kni`](https://www.nuget.org/packages/FlatRedBall2.Kni)
